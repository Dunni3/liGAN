# liGAN

liGAN is a Python codebase for structure-based drug discovery with deep generative models of atomic density grids.

## Dependencies

- numpy
- pandas
- scikit-image
- torch
- openbabel
- rdkit
- [molgrid](https://github.com/gnina/libmolgrid)
- [gnina](https://github.com/gnina/gnina)
